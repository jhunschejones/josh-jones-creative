# Josh Jones Creative — Landing Page

A clean, light-weight placeholder landing page for **Josh Jones Creative**, an upcoming independent multimedia project based in Tokyo, Japan.

## 🚀 Overview

This repository contains the single-page web portal serving as an informational hub for future creative operations. It outlines intended services across photography, audio/video production, and digital media consulting while awaiting official immigration approvals.

## 🛠️ Features

* **Tailwind CSS v4:** Utilizes the latest utility-first framework via CDN for modern styling.
* **Fully Responsive:** Optimized for mobile, tablet, and desktop viewports.
* **Compliance-focus:** Explicitly displays the project's current status regarding visa regulations to maintain total transparency with immigration authorities.
